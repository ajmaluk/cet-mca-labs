# Python Programming Lab

This directory contains Python scripts for the **CET MCA Semester 1** laboratory experiments. The labs are organized into **Cycles**, following the course syllabus progressively from basic syntax to advanced library usage.

## 📁 Course Structure

### [Cycle 1: Python Basics](./cycle%201/)
Fundamental concepts including variables, data types, conditions, and basic loops.
- **Key Experiments**: Leap Year, Area Calculations, List Comprehensions, Word Frequency.

### [Cycle 2: Functions and Data Structures](./cycle%202/)
Deeper dive into modular programming and built-in data structures.
- **Key Experiments**: Lambda Functions, Recursion, List operations.

### [Cycle 3: Modules and Packages](./cycle%203/)
Working with external and internal modules (Math, Time, Random).

### [Cycle 4: Object-Oriented Programming](./cycle%204/)
Class design, inheritance, and encapsulation in Python.

### [Cycle 5: Data Science Libraries](./cycle%205/)
Introduction to numerical processing and data manipulation.
- **Libraries used**: NumPy, Matplotlib.

---

## 🚀 Execution Guide
Ensure you have Python 3.x installed. Run any program using:
```bash
python3 filename.py
```


## 📝 Notes & Best Practices
- Use docstrings and comments to explain your code, especially for complex logic.
- Use try/except for user input and file operations to handle errors gracefully.
- Prefer list comprehensions and built-in functions for clarity and efficiency.
- Test your code with different input values, including edge cases.
- For data science cycles, ensure required packages (NumPy, Matplotlib) are installed:
	```bash
	pip install numpy matplotlib
	```
