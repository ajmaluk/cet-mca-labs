CSS Integration Methods

High Specificity

Inline CSS

Inline CSS is applied directly to specific HTML elements using the style attribute.

Key Characteristics:
  • Applied directly within the HTML tag.
  • Overrides styles from internal and external sheets.
  • Not recommended for large projects due to poor reusability.
Syntax Example: 
<h1 style="color: blue; font-size: 20px;">Hello World</h1>
Page Scoped

Internal CSS

Internal CSS is defined within a <style> block in the <head> section of an HTML document.

Key Characteristics:
  • Affects all elements of a specific type on a single page.
  • Useful for unique styles on standalone pages.
  • Increases the size of the HTML file.
Syntax Example: 
<style>
    body { background-color: #f0f0f0; }
    p { color: #333; line-height: 1.5; }
</style>
Best Practice

External CSS

External CSS is stored in a separate .css file and linked to the HTML document using a <link> tag.

Key Characteristics:
  • Maintain a single style file for multiple pages.
  • Faster performance due to browser caching.
  • Separates structure (HTML) from presentation (CSS).
Syntax Example: 
<!-- In HTML Head -->
<link rel="stylesheet" href="styles.css">

/* In styles.css */
.title {
    font-weight: bold;
    color: var(--primary-color);
}
