# CET MCA Semester 1 Lab Repository

Welcome to the comprehensive lab resource for **Master of Computer Applications (MCA)** at **College of Engineering Trivandrum (CET)**. This repository contains the complete source code, documentation, and implementation details for all Semester 1 laboratory courses.

## 🎓 Academic Context
- **Institution**: College of Engineering Trivandrum (CET)
- **Program**: Master of Computer Applications (MCA)
- **Semester**: 1
- **Focus Area**: Advanced Data Structures, Python Programming, and Web Technologies.

---

## 📂 Laboratory Courses

### 1. [Advanced Data Structures (ADS)](./ads/)
Implementation of fundamental and advanced data structures using C.
- **Key Concepts**: Linked Lists (Singly, Doubly, Circular), Stacks & Queues, Trees (BST, BFS/DFS), Graphs (Kruskal, Prim's), and Set Operations.
- **Documentation**: Includes algorithms and complexity analysis.

### 2. [Python Programming](./python/)
A series of laboratory experiments (Cycles) covering Python from basics to advanced data science libraries.
- **Key Topics**: Core Python Syntax, Data Structures, OOPs, NumPy, Pandas, and File Handling.
- **Structure**: Organized into Cycle 1 through Cycle 5.

### 3. [Web Programming](./web/)
Modern web development lab exercises focusing on responsive design and PHP-based dynamic web applications.
- **Technologies**: HTML5, CSS3 (Glassmorphism), JavaScript (ES6+), and PHP/MySQL.
- **Features**: Interactive UI components and functional database-driven projects.
